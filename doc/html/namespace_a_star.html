using AStar::HeuristicFunction = std::function<uint(Vec2i, Vec2i)>

Alias pour une fonction heuristique dans l'algorithme A*.

Une fonction heuristique prend deux paramètres Vec2i (coordonnées source et cible) et retourne un uint représentant l'estimation du coût pour atteindre la cible depuis la source.
